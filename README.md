## 💻 Sobre o projeto

O dtmoney é uma aplicação web para gerenciamento de finanças, a qual permite o cadastro de entradas e saídas.

---

## ⚙️ Tecnologias

- ReactJS
- Styled Components
- MirageJS
- Axios

---

## Rodando o projeto:

```bash
yarn

yarn dev
```
